## 2024-05-15 - Fast Array Allocation in `generateAllCandidates`
**Learning:** `generateAllCandidates` in V8 is a hot loop. Replacing dynamic `.push()` calls and `.length` increments on generic JavaScript Arrays with pre-allocated local `Array` and `Int32Array` usage for bounded numeric ranges avoids dynamic resizing penalties and V8 garbage collection overhead, making the routine execution ~40% faster.
**Action:** Identify array buildup inside large combinatorial searches (like DP preparation stages). When the exact or max length is easily calculable mathematically (e.g. `numWorkdays * allowance`), prefer creating a fixed-size array and manually incrementing a cursor (`cLen`), eventually truncating `array.length = cLen` over using `.push()`. Use `Int32Array` when specifically storing sequential integers like indices.
